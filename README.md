# Rezka.ag Universal Downloader

Универсальный скрипт на Python для автоматического скачивания фильмов и серий с сайта **rezka.ag** (и его зеркал) с обходом защиты Cloudflare и автоматической блокировкой рекламы.

## Требования

1. **Python 3.12+**
2. **Google Chrome** (установленный в системе)
3. **FFmpeg** (установленный и добавленный в системную переменную `PATH` или находящийся в корневой директории скрипта)

## Установка

1. Установите зависимости:
   ```bash
   pip install -r requirements.txt
   ```
   *Примечание: скрипт автоматически скачает и настроит расширение uBlock Origin Lite при первом запуске.*

## Использование

### 1. Интерактивный режим
Запустите скрипт без аргументов, и он пошагово запросит все параметры в консоли:
```bash
python downloader.py
```

### 2. С использованием аргументов (CLI)
Вы можете автоматизировать запуск, передав аргументы:
```bash
python downloader.py -u "[URL]" -t "[Озвучка]" -q [Качество] -s [Сезон] -e [Диапазон]
```

**Примеры:**
* Скачать сериал целиком (или выбрать параметры в процессе):
  ```bash
  python downloader.py -u "https://rezka.ag/animation/romance/40293-sonni-boy-2021.html"
  ```
* Скачать конкретную озвучку, разрешение и диапазон серий:
  ```bash
  python downloader.py -u "https://rezka.ag/animation/romance/40293-sonni-boy-2021.html" -t "AniLibria" -q 1080 -s 1 -e 1-3
  ```

## Ключевые возможности

### 🌐 Выбор режима работы браузера (Фоновый режим)
При обычном запуске скрипт автоматически спросит в консоли:
`Run browser in background (headless mode)? [Y/n] (default Y): `
* Нажатие **Enter** или ввод `y` / `yes` запустит браузер в фоновом режиме (окно Chrome не будет открываться).
* Ввод `n` / `no` запустит браузер в видимом режиме (полезно, если требуется визуально пройти сложную капчу от Cloudflare).
* Использование флага `--headless` в CLI аргументах сразу запускает скрипт в фоновом режиме без интерактивного вопроса.

### 🖥️ Выбор качества и реальные разрешения потоков
HDRezka использует сдвиг в маркировке разрешений потоков для бесплатных пользователей. Скрипт автоматически получает список доступных форматов и выводит в консоли меню выбора с указанием их физического разрешения:
* **`1080p Ultra (Premium / True 1080p)`** — полноценный поток 1920x1080 (требуется премиум-аккаунт, авторизованный в браузере, иначе скачивание ограничится 1-минутным превью).
* **`1080p (Free / 720p)`** — максимальное бесплатное качество (разрешение 1280x720). Рекомендуется по умолчанию.
* **`720p (Free / 480p)`** — разрешение 854x480.
* **`480p (Free / 360p)`** — разрешение 640x360.
* **`360p (Free / 240p)`** — разрешение 426x240.

### ⚡ Быстрое AJAX-получение потоков (Без кликов по плееру)
Для получения ссылки на поток скрипт больше не симулирует клики мыши по элементам интерфейса плеера (шестерёнке качества). Вместо этого он выполняет защищенный AJAX-запрос внутри Chrome с автоматической передачей cookie-файлов и декодирует полученную ссылку встроенным Base64-алгоритмом. Это гарантирует 100% надёжность работы независимо от изменений в вёрстке плеера.

### 📝 Автоматическое скачивание субтитров (Soft Subtitles)
Если у выбранного перевода (например, «Оригинал (+субтитры)») доступны внешние мягкие субтитры в формате WebVTT (`.vtt`), программа:
* Автоматически считывает доступные дорожки субтитров.
* Приоритетно скачивает дорожку на **русском языке** (`ru`), при её отсутствии — на **английском** (`en`), иначе — первую доступную.
* Сохраняет файл субтитров рядом с видео с аналогичным базовым именем (например, `downloads/[Название]/Season_1/[Название]_S1_Ep_1.ru.vtt`). Такие файлы автоматически распознаются большинством современных видеоплееров (VLC, MPC-HC и др.).

## Все опции запуска (CLI)

* `-u`, `--url` — ссылка на фильм или сериал.
* `-o`, `--output` — папка для сохранения файлов (по умолчанию `downloads`).
* `-t`, `--translator` — имя озвучки (подстрока без учета регистра, например `Дубляж` или `AniLibria`).
* `-q`, `--quality` — желаемое качество. Можно указать цифру (например `1080`, `720`, `480`, `360`), точное имя ключа (например `1080p`, `1080p Ultra`), либо `'best'` для лучшего бесплатного качества.
* `-s`, `--season` — номер сезона.
* `-e`, `--episode` — номер серии (например `5`) или диапазон серий (например `1-5`).
* `--headless` — принудительный запуск браузера в фоновом режиме без интерактивного вопроса.
